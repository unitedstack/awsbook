> ### **管理工具**

* Amazon CloudWatch
  
 监控资源和应用程序
  
* Amazon EC2 Systems Manager 
 
 配置和管理 EC2 实例和本地服务器
  
* AWS CloudFormation
  
 使用模版来创建和管理资源
  
* AWS CloudTrail
  
 追踪用户活动和 API 使用情况
  
* AWS Config
  
 追踪资源库存和变更
  
* AWS OpsWorks
  
 利用 Chef 自动化操作
  
* AWS Service Catalog
  
 创建和使用标准化产品
  
* AWS Trusted Advisor
  
 优化性能和安全性
  
* AWS Personal Health Dashboard
  
 AWS 服务运行状况的个性化视图
