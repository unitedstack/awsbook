> ### **数据库**

* **Amazon Aurora**

  高性能的托管关系数据库

* **Amazon RDS**

  适用于 MySQL、PostgreSQL、Oracle、SQL Server 和 MariaDB 的托管关系数据库服务

* **Amazon DynamoDB**

  托管 NoSQL 数据库

* **Amazon ElastiCache**

  内存中的缓存系统

* **Amazon Redshift**

  快速简单、经济高效的数据仓库

* **AWS Database Migration Service**

  以最少停机时间迁移数据库
